## MinMax标准化

MinMaxScaler方法

(N-MIN)/(MAX-MIN)

    import pandas as pd
    from sklearn.preprocessing import MinMaxScaler

    data = pd.read_excel('./jan.xlsx')

    scaler_2 = MinMaxScaler()

    data['A_Z'] = scaler_2.fit_transform(data['A'].values.reshape(-1,1))

    print(data['A_Z'])
